# 2026年3月を目途に企画２０個、プラグイン/コマンド/MODの制作物を２５個以上作る。

## - 企画作成実績 2/20個
### らーす鯖
１． **RMG鯖イベント企画・進行**

ミニゲームのRMG鯖が一時閉鎖するため、その前にみんなでRMGのミニゲームで遊ぶというイベントを企画した。ベッドウォーズやマーダーミステリーなどで遊んだ。また、最後にイベントの記念撮影を行った。３０人以上が訪れるイベントとなった。
![ibent](image.png)

２. **宣伝イベントの企画・進行**

らーす鯖をより多くのマインクラフターに知ってもらうため、Xを使用した宣伝イベントを企画した。目標はDiscord参加人数5000人、Xのフォロワー数500人を超えることだったが、結果は未達成で閉幕した。しかし、このイベントを通じて、Xを稼働させ、継続してポストを行っている。
